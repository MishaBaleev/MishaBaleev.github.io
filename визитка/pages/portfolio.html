Примеры проектов на ЯП LUA

Funny Rattle Lite

Развивающая игра для детей 4+ лет с поддержкой более чем 8 языков, созданная для платформы IOS, поддерживает и Iphone и Ipad. В данный момент игра расположена в App Store в виде двух версий, платной-полной и бесплатной-ограниченной.
В планах: оптимизировать приложение и улучшить его адаптивность.

Space Traveller

Игра для детей 6+ лет, созданная для платформы IOS, в данный момент поддерживает только Iphone. Сейчас игра не анонсирована из-за трудностей, связанных с созданием Provisioning Profile для реализации проекта в App Store.
В планах: добавить поддержку нескольких языков, адаптировать под Ipad, анонсировать в App Store.

Примеры проектов на ЯП Python

Telegram-бот 'Family'

Telegram-бот, написанный на ЯП Python Telebot, имеет возможность отправлять по запросу пользователя погоду в определенной местности, предоставлять базовую информацию по предоставленному IP-адресу, включает в себя краткую инструкцию с шаблонами вызовов и обратной связью.
В планах: увеличить функциональность бота, добавить историю переписки у бота.
